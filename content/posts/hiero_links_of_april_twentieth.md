## Hiero Links For The Week Of April 20th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of things that happend this past week. **Check it out!**

---

**Vedant** refactored the Hiero-Website [home page](https://github.com/hiero-ledger/hiero-website/pull/403) and optimized image load times to optimize performance, user experience, accessibility, and maintainability.

**Manmath Mohanty** added a return type annotation to the is [_payment_required](https://github.com/hiero-ledger/hiero-sdk-python/pull/2188) method in CryptoGetAccountBalanceQuery to maintain consistency across the Python SDK. 

**Michael Garber** corrected [draft HIP diagram rendering](https://github.com/hiero-ledger/hiero-improvement-proposals/pull/1451) and added a dark-theme image backdrop in Hiero Improvement Proposals to improve how Hip diagrams render on Hips.hedera.com.

**Janapareddy Vidya Varshini** added daily [scheduled builds](https://github.com/hiero-ledger/hiero-sdk-cpp/pull/1490) for both Windows and MacOs systems in the C++ SDK to surface build failures early and ensure a healthy matrix across all platforms.
