## Hiero Links For The Week Of April 27th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of things that happened this past week. **Check it out!**

---

**ivaylonikolov7** implemented High Volume Entity Creation [HIP 1313](https://github.com/hiero-ledger/hiero-sdk-js/pull/3974) in the JavaScript SDK to enable efficient batch creation of large numbers of entities. 

**Keshav Kumar Sharma** added a [good first issue guidelines](https://github.com/hiero-ledger/hiero-sdk-java/pull/2723) for the Java SDK to define what is and is not appropriate for beginner contributors. 

**Dosi Kolev** created an updated [release](https://github.com/hiero-ledger/hiero-sdk-go/pull/1689) of the Go SDK to document new changes and updates. 

**Matt Hess** updated [test constructor](https://github.com/hiero-ledger/hiero-consensus-node/pull/25198) invocation to correct a signature mismatch compilation error in Hiero Consensus Node.
