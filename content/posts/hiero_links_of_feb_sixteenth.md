## Hiero Links For The Week Of February 16th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---

**Angelina Ceppaluni** updated the JavaScript [Readme](https://github.com/hiero-ledger/hiero-sdk-js/pull/3703) to include helpful instructions on how to get started on a Windows machine.

**gangulysiddhartha22-cmyk** added module-specific [CodeRabbit review instructions](https://github.com/hiero-ledger/hiero-sdk-python/pull/1816) for the schedule module to improve automated review quality, speed, and focus on critical patterns such as anti-nesting safeguards, signature semantics, wait_for_expiry behavior, and protobuf fidelity.

**Hiero Good-First-Issue Team** launched their first set of [good-first-issues](https://github.com/hiero-ledger/hiero-sdk-js/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22Good%20First%20Issue%22) for the JavaScript SDK. 

**freya-docs** [removed sections](https://github.com/hiero-ledger/hiero-website/pull/219) about contributing and connecting with the community to streamline the content and improve overall clarity.

**Manish Dait** implemented [ScheduleInfoQuery](https://github.com/hiero-ledger/hiero-sdk-java/pull/2615) for the TCK, introducing support for schedule info queries. 

**Ivan Ivanov** implemented additional [TCK Endpoints](https://github.com/hiero-ledger/hiero-sdk-java/pull/2616) into the Java SDK.
