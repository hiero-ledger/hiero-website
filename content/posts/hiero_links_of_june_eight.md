## Hiero Links For The Week Of June 8th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of things that happened this past week. **Check it out!**

---
**Manish Dait** added the [getAccountBalance](https://github.com/hiero-ledger/hiero-sdk-python/pull/2336) method to the Python SDK TCK module.

**Fantomasa** corrected a flaky [TopicMessageMocking](https://github.com/hiero-ledger/hiero-sdk-js/pull/4140) completion check in the JavaScript SDK to ensure stream completion is detected reliably on the CI.

**Mustafa Uzun** released a [version update](https://github.com/hiero-ledger/hiero-sdk-java/pull/2781) of the Java SDK.

**Peter Chen J.** updated [TestUnitConcurrentGetNodeReadmit](https://github.com/hiero-ledger/hiero-sdk-go/issues/1714) in the Go SDK to correct a Windows Panic Issue.

**Angelina Ceppaluni** updated [workflows](https://github.com/hiero-ledger/hiero-enterprise-java/pull/208) in Enterprise-Java to use updated SHA keys.
