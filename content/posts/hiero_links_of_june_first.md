## Hiero Links For The Week Of June 1st

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---
**Anshul** added a blog post for [HIP-1195: Hiero Hooks and an Application to Allowances](https://github.com/hiero-ledger/hiero-website/pull/466) to the Hiero Website.

**Daniel Ntege** added the [updateAccount](https://github.com/hiero-ledger/hiero-sdk-python/pull/2320) endpoint in the Python SDK to support updating accounts via TCK endpoint. 

**Angelina Ceppaluni** added [coderabbit review instructions](https://github.com/hiero-ledger/hiero-enterprise-java/pull/185) to the Hiero-Enterprise-Java base module to assist in conducting PR reviews. 

**Mustafa Uzun** updated the [protofiles](https://github.com/hiero-ledger/hiero-sdk-java/issues/2765) to version 0.74 in the Java SDK.
