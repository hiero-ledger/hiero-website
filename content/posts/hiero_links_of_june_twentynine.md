## Hiero Links For The Week Of June 29th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---
**[Daniel Ntege](https://github.com/danielmarv)** updated the [undici version](https://github.com/hiero-ledger/hiero-website/pull/519) in the Hiero Website to address six security vulnerabilities. 

**[Blaze1425](https://github.com/blaze1425)** updated [workflows](https://github.com/hiero-ledger/hiero-sdk-python/pull/2381) in the Python SDK to standardize github action version pins.

**[Xin Li](https://github.com/xin-hedera)** fixed the lazy [database pool](https://github.com/hiero-ledger/hiero-mirror-node/pull/13814) connection issue in rosetta inside the Hiero Mirror Node Repository. 

**[deshmukhpranali](https://github.com/deshmukhpranali)** updated the [GitBook sync workflow](https://github.com/hiero-ledger/hiero-block-node/pull/3128) in Hiero Block Node to push documentation changes to a sync branch, open or update pull requests instead of pushing directly to master, and sign commits to satisfy DCO requirements.
