## Hiero Links For The Week Of June 22nd

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---
**Daniel Ntege** removed a [ref configuration](https://github.com/hiero-ledger/hiero-sdk-python/pull/2) in the Python SDK workflow to simplify the workflow and rely on default behavior for commit checkout.

**Fantomasa** upgraded [@grpc/grpc-js from 1.12.6 to 1.12.7](https://github.com/hiero-ledger/hiero-sdk-js/pull/4171) and regenerated the pnpm lockfile to address two high-severity security advisories identified by Snyk in the JavaScript SDK.

**Noah Tayebwa** added support for [deleteAllowance](https://github.com/hiero-ledger/hiero-enterprise-java/pull/217) in the Hiero Enterprise Java repository.

**Jessica G** added a [blog post](https://hiero.org/blog/hiero-enterprise-java-v0200-release) to highlight the release of Hiero Enterprise-Java.
