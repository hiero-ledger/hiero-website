## Hiero Links For The Week Of March 16th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---
**Sophie Bulloch** released an updated version of the [Python SDK](https://github.com/hiero-ledger/hiero-sdk-python/pull/1993) to provide insights into recent updates and changes. 

**Pavel Borisov** updated the C++ SDK by replacing several GitHub Actions with their [StepSecurity-maintained](https://github.com/hiero-ledger/hiero-sdk-cpp/pull/1225) counterparts to improve security and maintainability.

**Giuseppe Bertone** updated the JavaScript SDK by migrating its dependency from [ethers v5](https://github.com/hiero-ledger/hiero-sdk-js/pull/3870) to ethers v6, addressing known vulnerabilities and risks.

**Nguyen Kim Hieu** updated the Java SDK [test suite](https://github.com/hiero-ledger/hiero-sdk-java/pull/2663) by replacing wildcard imports with explicit imports in CustomFeeLimitTest, improving code clarity and maintainability.
