## Hiero Links For The Week Of May 4th

This week featured a lot of new and exciting updates within the Hiero Organization. 
Below are links to some of the things that happened this past week. **Check it out!**

---
**Mohit Sharma** updated the [TopicUpdateTransaction](https://github.com/hiero-ledger/hiero-sdk-python/pull/2241) class to accept the generic Key interface for all key-related fields in the Python SDK.

**Jessy Ssebuliba** and **Angelina Ceppaluni** added a blog post to the Hiero Website outlining the success of the [Open Source event](https://hiero.org/blog/kampala-hiero-event) in Kampala, Africa.

**Rob Walworth** created an updated [release](https://github.com/hiero-ledger/hiero-sdk-swift/pull/624) of the Swift SDK to outline the new features and improvements.

**Chaitanya Medidar** updated the [bot-on-comment](https://github.com/hiero-ledger/hiero-sdk-cpp/pull/1581) workflow in the C++ SDK to dispatch slash commands through a command_handlers map instead of a manual if/else chain.

**Alfredo Gutierrez** added[ end-to-end regression tests](https://github.com/hiero-ledger/hiero-block-node/pull/2705) for stalled publisher handling and correct ACK ordering, validating RESEND behavior and improving CI reliability across the messaging pipeline.
