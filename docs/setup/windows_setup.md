# Windows Setup Guide for Hiero Website

This guide provides a single, linear set of steps to set up the Hiero Website locally on a Windows machine.

---

## Prerequisites

Install the following tools before proceeding:

Git for Windows: https://git-scm.com/download/win  
Node.js (LTS): https://nodejs.org/  
Visual Studio Code (recommended): https://code.visualstudio.com/

---

## Verify Installations

Open PowerShell or Git Bash and run the following commands:

git --version
node -v
